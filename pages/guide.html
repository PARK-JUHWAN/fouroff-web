Guide

사용 가이드

포오프 시작하기

  • 수간호사
  • 간호사
1

방 만들기

앱 실행 → '방 만들기' 클릭 → 병동 이름과 비밀번호 입력

2

기본정보 입력

연/월, 간호사 수, 근무 유형(D/E/N) 설정

3

과거 3일 입력

이전 달 마지막 3일 근무 입력 (연속근무 방지용)

4

근무조건 설정

주말/평일 인원, N 개수, 저연차 방지 등 병동 규칙에 맞게 설정

5

방 정보 공유

방 이름과 비밀번호를 간호사들에게 전달하세요

6

근무표 생성

희망근무 마감 후 '생성' 버튼 클릭으로 최적의 근무표 완성

1

방 입장

수간호사님께 받은 방 이름과 비밀번호를 입력하세요

2

희망근무 입력

달력에서 원하는 날짜를 터치하고 D/E/N/X 중 선택하세요

3

결과 확인

수간호사님이 생성을 완료하면 결과를 확인할 수 있습니다

지금 바로 시작하세요
