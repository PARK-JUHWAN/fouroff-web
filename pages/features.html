Auto Generate

버튼 하나로 완성되는 근무표

모든 제약조건을 만족하는 최적의 근무표를 수학적 알고리즘이 자동으로 생성합니다. 더 이상 엑셀과 씨름하지 마세요.

  • 수학적 최적화 알고리즘 (Constraint Satisfaction)
  • 모든 제약조건을 만족하는 최적해 도출
  • 10~30초 내 생성 완료

Preference Input

터치 한 번으로 희망 입력

달력에서 원하는 날짜를 터치하고 D/E/N/X 중 선택하면 끝. 같이 근무하고 싶지 않은 동료도 설정 가능합니다.

  • 직관적인 달력 UI
  • D/E/N/X 간편 선택
  • 기피 동료 설정 가능

Fair Distribution

수학적으로 균등한 연휴 배분

모든 간호사에게 공평한 4연휴를 배분합니다.
특정인에게 편중되지 않는 투명한 알고리즘.

Work Conditions

병동 상황에 맞는 세밀한 설정

주말/평일 근무자 수, 나이트 개수, 저연차끼리 방지 등 우리 병동만의 규칙을 설정하세요.

  • 주말/평일 근무자 수 설정
  • 월별 N 근무 횟수 조절
